Help 帮助中心

1. Getting Started / 开始使用

Step 1: Get Device ID / 获取设备ID
On first open, the authorization screen shows your 8-digit Device ID (e.g., A3B9K7P2). Click to copy.
首次打开工具时出现授权界面，显示您的 8位设备ID（如 A3B9K7P2）。点击即可复制。

Step 2: Request Authorization / 申请授权
Send the Device ID to your CANNY sales rep to receive the authorization code.
将设备ID发送给康力电梯销售代表，等待接收授权码。

Step 3: Enter Code / 输入授权码
Paste the code and click "Verify". You will enter main screen automatically upon success.
粘贴授权码到文本域，点击"Verify / 验证授权"。验证成功后自动进入主界面。

Important / 重要提示
The code is bound to this device and browser. Reapply if you switch browsers or clear cache.
授权码与当前设备ID及浏览器绑定。更换浏览器或清理缓存需重新申请。

5. Load Factor / 装载系数

Auto / 自动（默认）
Starts at 80% and decreases until both targets are met with min. elevators.
从 80% 开始递减，直到同时满足输送能力与间隔目标，且电梯数量最少。

Manual / 手动
Select any value 30% – 80% from the dropdown. Click "Resume Rec. value" to switch back to auto.
可从下拉框选择 30% – 80% 任意值。点击 "恢复推荐值" 切回自动。

Tip / 提示
Higher load factor → more crowded car → higher handling capacity, but larger average interval.
装载系数越高 → 轿厢越拥挤 → 输送能力越高，但平均间隔越大。
